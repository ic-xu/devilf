## 引擎介绍 

引擎主页: https://ymbok.com
适用范围：中小型游戏，APP内营销活动，小游戏。  
主要特性：游戏循环，精灵动画，交互控制。  
游戏支持：Tiled地图，Plist精灵，碰撞检测，遥杆控制器等。
